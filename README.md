# Realmsz

![Logo](public/cover.jpg)

## Overview

Realmsz is a comprehensive cloud-based platform that combines 10 essential modules in one place:
- Focus: Task and project management
- Plans: Strategic planning and goal setting
- Profile: Personal and professional profiles
- Network: Community and social connections
- Milai: AI-powered assistance
- Studio: Content creation and management
- Finance: Financial tracking and management
- Crypto: Cryptocurrency and NFT integration
- Websites: Website management and hosting
- Storage: Secure file storage and sharing

## Features

- **Integrated Modules**: Seamless integration between all platform modules
- **AI-Powered**: Advanced AI capabilities through Milai
- **NFT Integration**: Support for NFT creation and management
- **Secure Storage**: Encrypted file storage and sharing
- **Community Features**: Social networking and collaboration tools
- **Financial Tools**: Comprehensive financial management
- **Content Creation**: Advanced studio tools for content creation

## Technology Stack

- **Backend**: Laravel 10
- **Frontend**: Vue.js 3, Inertia.js
- **Styling**: TailwindCSS
- **Database**: MySQL
- **Authentication**: Laravel Sanctum
- **AI Integration**: Custom AI solutions
- **NFT Support**: Integration with dracoscopia.com

## Requirements

- PHP 8.1+
- Node.js 16+
- MySQL 5.7+
- Composer
- NPM/Yarn

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/realmsz.git
cd realmsz
```

2. Install PHP dependencies:
```bash
composer install
```

3. Install JavaScript dependencies:
```bash
npm install
```

4. Create environment file:
```bash
cp .env.example .env
```

5. Generate application key:
```bash
php artisan key:generate
```

6. Configure database in .env file

7. Run migrations:
```bash
php artisan migrate
```

8. Build assets:
```bash
npm run build
```

9. Start the development server:
```bash
php artisan serve
```

## Configuration

- Configure email settings in `.env` for email verification
- Set up NFT integration with dracoscopia.com
- Configure storage settings for file uploads
- Set up AI integration for Milai module
